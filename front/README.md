# Système de vote électronique - Client (Front-end)

## Technologies utilisées
- TypeScript
- React
- TailwindCSS
- Vite
- yarn (équivalent de npm)
- chartJS (graphiques)

## Installation

1. Installer yarn :
```bash
npm install -g yarn
```
2. Installer les dépendances :
```bash
yarn
```
3. Lancer le serveur de développement :
```bash
yarn dev
```
